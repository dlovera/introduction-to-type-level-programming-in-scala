Learnings

  • Type Level Programming is not that different!
  • Instead of values we produce types.
  • There are 2 types of parameters: value and type parameter.
  • With Types you can constrain your program to become more correct.
@notesMd { - elevated my mind beyond Java Level generics to tell the compiler what's inside a container - Scala has two parameter lists: one at the value level and one at the type level - you enhance your program with type level metadata to constrain your program. - Many or all exceptions can be replaced if you model your program accordingly (example: NonEmptyList) }
